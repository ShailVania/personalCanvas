# Shail Vania | Personal Portfolio

## Description

Welcome to my personal portfolio website! This project serves as a dynamic online resume, showcasing my skills, projects, and contact information. Built with Next.js, the site is designed to be fast, responsive, and easy to navigate.

## Features

*   **Project Showcase:** Detailed descriptions and links to my personal and professional projects.
*   **Skills Section:** A comprehensive list of technologies and tools I am proficient in.
*   **About Me:** A brief introduction and overview of my background and experience.
*   **Contact Form:** A simple way for visitors to get in touch with me.
*   **Responsive Design:** Optimized for viewing on various devices (desktops, tablets, and mobile phones).
*   **Theme Switching:** Allows users to toggle between light and dark modes.

## Technologies Used

*   Next.js
*   React
*   TypeScript
*   Tailwind CSS
*   shadcn/ui
*   vercel

## Setup Instructions

1.  Clone the repository: `git clone <repository_url>`
2.  Install dependencies: `npm install` or `yarn install`
3.  Run the development server: `npm run dev` or `yarn dev`
4.  Open your browser and visit `http://localhost:3000`
